# SmartConfig Frontend

Frontend del configurador de robots industriales SmartConfig, construido con **React + TypeScript + Vite**.

## Características

- ✅ Configuración de robots por industria o parámetros técnicos
- ✅ Selección de familias, modelos y accesorios
- ✅ Validación de compatibilidades y exclusiones
- ✅ Cálculo automático de precios
- ✅ Exportación y envío de configuraciones
- ✅ Estado global con Zustand
- ✅ Estilos con Tailwind CSS
- ✅ Consumo de API Strapi v5

## Requisitos

- Node.js >= 18
- npm >= 6
- Backend de Strapi corriendo en `http://localhost:1337`

## Instalación

```bash
cd smartconfig-frontend
npm install
```

## Scripts Disponibles

```bash
# Desarrollo (puerto 3000)
npm run dev

# Build para producción
npm run build

# Preview del build
npm run preview

# Linting
npm run lint
```
